## Troubleshooting

### Resume not parsing correctly?
- Ensure resume has clear sections (Experience, Education, Skills)
- Use standard formatting
- PDF works best

### Template not displaying?
- Check browser console for errors
- Try refreshing the page
- Clear browser cache

### Deployment issues?
- Make sure HTML file downloaded completely
- Check file size (should be under 1MB)
- Verify internet connection for external resources
