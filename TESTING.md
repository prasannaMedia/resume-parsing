## How to Test

1. Create a PDF or Word document with the above content (or use your own resume)
2. Start the development server: `npm run dev`
3. Open the application in your browser
4. Upload your resume file
5. Select a template
6. Preview and download your portfolio

## What to Expect

The application will:
- Parse your resume and extract key information
- Display it in the chosen template
- Allow you to download a standalone HTML file
- Provide deployment instructions for Netlify and GitHub Pages
